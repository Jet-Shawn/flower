# Memory Garden Workshop

Welcome! In this workshop, you'll turn your personal memories into AI-generated flower art.

---

## Before You Start

Please complete the following steps **before** the workshop begins:

### 1. Download the project code

- Go to: [PASTE_YOUR_GITHUB_REPO_LINK_HERE]
- Click the green **Code** button → **Download ZIP**
- Unzip the downloaded file to a location you can easily find (e.g., your Desktop)

### 2. Download your memory file

- Download the memory file from here: [PASTE_MEMORY_FILE_LINK_HERE]
- Place the downloaded file into the unzipped project folder

### 3. Open the project in Cursor

- Open **Cursor** (download it from https://www.cursor.com if you haven't already)
- Go to **File → Open Folder**, then select the unzipped project folder

### 4. Follow the tutorial

- Open the tutorial directly on GitHub: [PASTE_YOUR_GITHUB_REPO_LINK_HERE]/blob/master/TUTORIAL.md
- Follow the steps starting from **Step 1** (you've already completed Step 0 by downloading the project)

### 5. API Key

Use the following API key when setting up your `.env` file (see Step 2 in the tutorial):

```
REPLICATE_API_TOKEN=PASTE_YOUR_API_KEY_HERE
```
